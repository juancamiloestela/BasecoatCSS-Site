Sections

Typography in Basecoat is king. All elements are carefully styled to match vertical rythms and ensure optimum readability. All dimensions are handled in ems so typography controls the dimensions of all other elements, even if you extend Basecoat to use other font families. All containers are targeted to have a max-width of 50em, this way each line of text will be the ideal width so that users can easily find the beginning of the next.

By default the font family used is sans-serif but you can easiy change the font in the <body> element and all elements will update accordingly.

Inline Elements

Inline elements are all tested to maintain the vertical rythm, even new html5 input controls.

Inline Elements

This is a paragraph to demonstrate all the inline elements with styles. For example this is a link. An this is strong text which should look the same as bold text. You can also emphasize text or make it italic. If you deleted content you can show it as strikethrough text.

Text can also have abbreviations like FBI or acronyms like ASAP. You can even show Sub text and Sup Text.


<p>This is a paragraph to demonstrate all the inline elements with styles. For example this is <a href="#">a link</a>. An this is <strong>strong text</strong> which should look the same as <b>bold text</b>. You can also <em>emphasize text</em> or make it <i>italic</i>. If you deleted content you can show it as <strike>strikethrough text</strike>.</p>

<p>Text can also have abbreviations like <abbr title="Federal Bureau of Investigation">FBI</abbr> or acronyms like <acronym title="as soon as possible">ASAP</acronym>. You can even show <sub>Sub</sub> text and <sup>Sup</sup> Text.</p>

Badges

Badges are a great way to give extra information to your users in a concise and efficient manner. In Basecoat badges align coherently with surrounding content and can have a circular class for rounded borders. Badges with class tag will get a pointed left edge like a real life tag and will degrade gracefully on older browsers to look like a squared paper tag.

Icons

Basecoat's aim is to serve as a robust and graphically coherent starting point for your app or website. It can be used by itself for a minimalistic look but you are encouraged to extend the base styles and create your own coats. When it comes to icons we understand there are trillions of icons out there, each with their own particular icon set and style. This is why we do not include an icon set with Basecoat. If we did, you would most likely have to include another iconset to meet your specific needs.

There are several very high quality icon sets available, some of which we have tried and work wonderfully with Basecoat are:

  • FontAwesome
  • Fontello
  • Typicons
