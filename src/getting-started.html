Getting Started From zero to css hero in 1 min

  • Install
  • Usage
  • Extend

Install

  • Using bower
    bower install --save basecoat-css
  • Cloning github repo
    git clone https://github.com/juancamiloestela/BasecoatCSS
  • Or download as a zip file.

Usage

  1. Include the basecoat css file on the <head> of your page 
    
		...
		<link rel="stylesheet" href="path/to/basecoat.min.css">
		<link rel="stylesheet" href="css/styles.css">
	</head>
  2. Include the basecoat js file after jQuery and before your custom site scripts 
    
		...
		<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
		<script src="path/to/basecoat-css.min.js"></script>
		<script src="js/script.js"></script>
	</body>
  3. Pat yourself on the back for a job well done!

Right now you can start coding your page. All js code is auto-initializable so you don't have to do a thing. Your site will be using the default styles provided by Basecoat which will provide a minimalistic style and beautiful typographical balance. But Basecoat is just, the base coat, on top of which you are encouraged to make your art either by hand or using coats made by others.

Extend & Customize

Basecoat's default styles are beautifully minimalistic and just work out of the box. But lets face it, we all want to put our personal style into our work. We are all artists. This is why basecoat has focused on being as minimal, yet as complete as possible. Keeping a low specificity index so that overriding styles does not become a nightmare, that means concise selectors and no !importants! ... !!

You can do this via two methods:

Coats Known in the dark world as plugins

Coats are pre-made files designed specifically to alter all the default styles coherently and for one purpose only. Think of coats as css plugins. Try some coats below. Mix them and see how this site will change on the fly.
